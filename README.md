# Task 1 - Exploring and Visualizing the Iris Dataset

## Objective
To explore and visualize the Iris dataset and understand feature distributions, relationships, and outliers.

## Dataset
Iris Dataset loaded using Seaborn.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Dataset inspection
- Summary statistics
- Missing value analysis
- Scatter plots
- Histograms
- Box plots
- Correlation heatmap

## Key Findings
- Dataset contains 150 samples and 5 features.
- No missing values.
- Strong correlation between petal length and petal width.
- Setosa species is clearly distinguishable from the others.
- Petal-related features are most informative.
